### Denis Kleptsov

Building offline, privacy-first ML tooling for the desktop —
document translation, PII redaction, agentic code assistants.

Currently focused on Apple Silicon: shipping native macOS apps powered by
quantised local LLMs (Hy-MT 1.5, Whisper, BitNet-class models).

---

#### Now

- **[Calque](https://github.com/dkleptsov/calque)** — offline MS Office translator for macOS (Tauri 2, 1.25-bit Hy-MT).
- **[DocLingua](https://github.com/dkleptsov/doclingua)** — sibling experiment in the same problem space.
- **PII Privacy Filter** — local, on-device PII redaction for macOS ([privacy policy](https://github.com/dkleptsov/pii_filter_mac_privacy)).

#### Selected work

- **[auto-debug-agent](https://github.com/dkleptsov/auto-debug-agent)** — autonomous Python agent that writes code, catches its own errors, self-corrects.
- **[agentic_rag](https://github.com/dkleptsov/agentic_rag)** — agentic RAG QA system on LangChain.
- **[char_diff_tool](https://github.com/dkleptsov/char_diff_tool)** — character-level diff utility for translated documents.
- **[server_setup](https://github.com/dkleptsov/server_setup)** — opinionated Linux server bootstrap scripts.
